WhatsApp Assistant Pro+ 隐私政策

最后更新日期：2024年12月9日

感谢您使用 WhatsApp Assistant Pro+。我们非常重视您的隐私和个人信息安全。本隐私政策旨在向您说明我们如何收集、使用和保护您的信息。

1. 信息收集

我们收集的信息仅限于：

  • 您在使用翻译功能时输入的文本内容
  • 您配置的 API 密钥（存储在本地浏览器中）
  • 基本的使用统计数据（如功能使用频率）

2. 信息使用

我们收集的信息仅用于：

  • 提供翻译和 AI 分析服务
  • 改进产品功能和用户体验
  • 解决技术问题和提供支持

3. 信息安全

我们采取以下措施保护您的信息：

  • 所有 API 密钥均使用浏览器的安全存储机制保存
  • 翻译内容仅在使用时临时处理，不会永久存储
  • 不会收集或存储您的 WhatsApp 聊天记录
  • 所有数据传输均采用加密方式

4. 第三方服务

我们使用以下第三方服务：

  • DeepSeek AI API（用于翻译和分析）
  • 通义千问 API（可选的翻译服务）
  • 火山翻译 API（可选的翻译服务）

这些服务的使用受其各自的隐私政策约束。我们建议您查看这些服务的隐私政策。

5. 用户权利

作为用户，您拥有以下权利：

  • 随时删除已保存的 API 密钥
  • 选择使用或禁用任何功能
  • 要求获取关于您的数据使用情况的信息

重要提示：

本扩展程序不会：

  • 存储或传输您的 WhatsApp 聊天记录
  • 收集您的个人身份信息
  • 与第三方共享您的使用数据
  • 在未经您同意的情况下使用您的信息

6. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的版本将在本页面发布，并在扩展程序中通知用户。继续使用我们的服务即表示您同意新的隐私政策。
