## ✨ 特性

- 🎯 实时消息翻译
- 🤖 AI 对话分析
- 💬 输入框实时翻译
- 🔄 多种翻译引擎支持
- 🎨 优雅的 UI 设计
- ⚡ 高性能，低延迟

## 🚀 快速开始

1. 从 [Chrome 网上应用店](https://chrome.google.com/webstore/pending) 安装扩展
2. 打开 [WhatsApp Web](https://web.whatsapp.com/)
3. 配置你的 API Key（可选）
4. 开始使用翻译和 AI 分析功能

## 🛠️ 支持的翻译服务

- [DeepSeek AI](https://deepseek.com/)
- [通义千问](https://tongyi.aliyun.com/)
- [火山翻译](https://translate.volcengine.com/)
